This WordPress Plugin transforms children pages into OnePager sections of their respective parent page.

== Installation ==

1. Upload 'plugin-onepager.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Note: The minimum PHP version required for this plugin is 5.3.0!
